# Galfus Script for Visual Studio Code

Language support for [Galfus Script](https://galfus.com/).

## Current features

- Galfus Script file recognition
- Syntax highlighting
- Comment and bracket configuration
- Automatic indentation

## Supported files

- `.gfs` — Galfus Script source files

## Development

Install the project dependencies with Bun:

```bash
bun install
```
