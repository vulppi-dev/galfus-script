# Galfus Script development

For any task involving Galfus Script source (`.gfs`), its syntax, type system,
ownership, builtins, workspace configuration, or the Rust frontend/compiler/VM,
read and follow `.agents/skills/galfus-script-development/SKILL.md` before making a
change.

Load the matching document from `.agents/skills/galfus-script-development/references/`
for language syntax, semantic rules, or workspace/builtin behavior. Treat the
parser, type validator, builtin source, and tests as the authority when they
conflict with a design document.
